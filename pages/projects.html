Modest (Pelican Theme)

Link: http://github.com/mandaris/modest
A simple Pelican theme that that attempts to integrate opengraph and schema.org tags as well as having an easy to read experience. It's what this blog is currently using.

Pelican Test Content

Link: https://github.com/mandaris/Pelican-Test-Content
Simple pelican articles to test new themes and pluggins quickly.

pelican_json_feed

Link: https://github.com/mandaris/pelican_json_feed A fork of the original because I was running into a number of issues that I couldn't understand from the original code. It's supposed to allow for generation of a json feed of a pelican blog without additional changes to what you have.

TuftePelican

Link: https://github.com/mandaris/TuftePelican
A pelican theme that attempts to use the Tufte CSS provide by Dave Liepmann. I hope I do him justice. I don't work on this anymore.
